SCANNING A QR CODE
1. Select a camera from the dropdown.
2. Press the "Prime Scanner" button.
3. Press the "Turn on Scanning Camera" button.
4. Scan the QR code with the camera.
5. Compare the Item code with the one displayed on the screen.
6. Refresh the camera by pressing the "Prime Scanner" button.
7. Repeat steps 4 to 6.
